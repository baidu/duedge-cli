**命令**
 delete-trigger

**功能说明**
 删除某条具体的触发器 

**返回参数**
+--------------+----------+------------------+
| 参数名称     | 参数类型 | 参数说明         |
+--------------+----------+------------------+
| errors       | list     | 错误信息         |
| request_uuid | string   | 请求uuid         |
| result       | dict     | 资源详情         |
| success      | bool     | 请求成功与否标志 |
+--------------+----------+------------------+

**返回结果**
{
	"errors": [],
	"result": true,
	"success": true
}
